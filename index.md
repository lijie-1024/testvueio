## Build Setup

``` bash


# 克隆到本地
git clone https://github.com/landluck/lucky_wheel.git

# 进入文件夹
cd lucky_wheel

# 安装依赖
npm install

# 开启本地服务器localhost:8088
npm run dev

# 发布环境
npm run build


#然后静待你的浏览器打开一个网页，按F12 进入手机模拟器，查看效果最佳(推荐chrome浏览器，前端开发者的必备)

#对你有帮助的话，请 star 一下
```
